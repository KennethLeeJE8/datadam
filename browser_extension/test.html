MCP Extension Test Page

Use this page to test the MCP Personal Data Autofill extension. Open the extension popup and click "Get Profiles" to fetch data from your MCP server.

Personal Information Form

Extension Test Instructions

  1. Load this extension in Chrome Developer Mode
  2. Click on the extension icon in the toolbar
  3. Click "Get Profiles" to fetch data from MCP server
  4. Click on any profile in the popup to autofill the form
  5. Use "Re-scan Fields" to detect form fields
  6. Use "Fill All Fields" to autofill with the first available profile
